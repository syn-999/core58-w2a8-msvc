# Code of Conduct

This project aims to be technically rigorous, respectful, and practical.

## Expected Behavior

- Be respectful and professional.
- Focus feedback on code, behavior, and outcomes.
- Assume good intent while still being precise about problems.
- Keep discussion grounded in reproducible technical details.

## Unacceptable Behavior

- Harassment or personal attacks
- Discriminatory or abusive language
- Deliberate disruption of project discussion
- Publishing private information without permission
- Repeated bad-faith engagement

## Enforcement

Project maintainers may edit, remove, or moderate content that violates this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the repository owner through GitHub.
