# Support

`core58-w2a8-msvc` is maintained on a best-effort basis.

## Where to Ask for Help

- Use GitHub Issues for reproducible bugs and concrete feature requests.
- Check the README and release notes first, especially for setup, packaging, and runtime expectations.

## What to Include

Please include as much of the following as possible:

- repo commit or release tag
- operating system version
- Python version
- CPU and GPU model
- exact command you ran
- model path and model name
- expected behavior
- actual behavior
- relevant logs or screenshots

## Good Support Requests

Good reports are narrow, reproducible, and use commands that someone else can run locally.

## What Support Does Not Cover

- private model distribution
- custom hardware tuning for unshared environments
- recovery of local data or weights
- debugging modified local trees without enough detail to reproduce the issue
